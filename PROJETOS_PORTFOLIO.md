**Categoria:** Mobile & IA

- Insights automáticos via IA
- Categorização automática de transações

### Tecnologias
- Flutter
- Dart
- Firebase
- Firestore
- Gemini AI
- Google Sheets API
- Google App Scripts

### Principais Conquistas
- Desenvolvimento de aplicativo mobile multiplataforma com Flutter
- Integração com planilhas Google via API
- Sincronização de dados em tempo real com Firestore
- Implementação de IA para análise financeira
- Automação de categorização de gastos com Gemini AI
- Geração de insights e relatórios financeiros

### Resumo Esperado
Um aplicativo mobile completo de controle financeiro que utiliza IA para automatizar a categorização de gastos e fornecer insights inteligentes sobre as finanças pessoais do usuário.

## Instruções para Criação dos Repositórios

1. **Criar repositório no GitHub** com o nome do projeto
2. **Adicionar README.md** com a descrição completa do projeto
3. **Incluir código fonte** organizado e comentado
4. **Adicionar documentação** de instalação e uso
5. **Configurar GitHub Pages** se aplicável
6. **Atualizar portfolio** com o link do repositório

## Notas Importantes

- Todos os projetos devem ter documentação clara
- Código deve estar bem comentado e organizado
- Incluir instruções de instalação e configuração
- Adicionar screenshots ou demos quando possível
- Manter repositórios atualizados e ativos
